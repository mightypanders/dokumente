# Tunnelprotokolle
## IPSec
"Standard" für gewerbliche Nutzung 
- proprietäre Nutzung
- Problem NAT
- Layer 3
## SSTP 
Ablösung für PPTP
- Layer 2
## PPTP
RC4 als Verschlüsselungsalgorithmus
- Keine Empfehlung
- Layer 2
## L2TP
Layer 2 Tunneling Protocoll 
- Tunnelaufbau, Authentifizierung 
- Keine Verschlüsselung
## IKE
Internet Key Exchange 
- Unterstandard von IPSec
- Layer 3
