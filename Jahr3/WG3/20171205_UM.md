# Netzplantechnik

0. Skizze
    - Um Überschneidungen zu erkennen und zu eliminieren
1. 
    1. Reinzeichnung
        - Übernahme der Skizze mit Formvorschrift  
    2. Vorwärtsrechnung
        - Prämisse:  
        ```
            FAZ Start = 0  
            FEZi = FAZi + di  
            FAZj = MaxFEZi  
        ```
    3. Rückwärtsrechnung
        - Prämisse:  
        ```
            FEZ Ende = SEZ Ende  
            SAZj = SEZj - dj  
            SEZi = Min SAZj  
        ```
    4. Puffer
        - Prämisse:
        ```
            GP = SEZi - FEZi  
            oder  
            SAZi - FAZi  
        ```
    5. Kritischer Pfad  
        - Verbindung aller Vorgänge mit GP = 0
